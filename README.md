## About

CineVault is a full-featured movie discovery web application built with React.js, powered by The Movie Database (TMDb) API. Users can search for any movie by title, explore real-time trending films (daily or weekly), and filter results by genre, release year, and rating. Clicking any movie opens a detailed modal showing the synopsis, runtime, genres, TMDb score, and an embedded YouTube trailer. Users can save movies to a personal Watchlist that persists across sessions using localStorage.
Tech used: React, Axios, TMDb API, CSS (custom design system), localStorage
